# ASR Portfolio Website

A responsive UI/UX designer portfolio landing page built with HTML, CSS, and JavaScript.

## Files

- `index.html` - page structure and content
- `styles.css` - layout, colors, typography, and responsiveness
- `script.js` - reveal-on-scroll interactions

## Customize

Update the bio, project descriptions, and social media links in `index.html` to match your real portfolio details.
